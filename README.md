# 🐍 Snake, Water, Gun Game (Python)

This is a simple **Snake, Water, Gun** game built using **Python**, where the user plays against the computer.  
The computer randomly chooses an option, and the winner is decided using game rules.

---

## 🎮 Game Rules

| Your Choice | Computer Choice | Result |
|------------|----------------|--------|
| Snake 🐍 | Water 💧 | You Win |
| Water 💧 | Gun 🔫 | You Win |
| Gun 🔫 | Snake 🐍 | You Win |
| Same Choice | Same Choice | Draw |
| Otherwise | — | You Lose |

---

## 🛠️ Technologies Used

- Python 🐍
- `random` module
